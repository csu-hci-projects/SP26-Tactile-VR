# SP26-Tactile-VR# Learning Fine Motor Skills in Virtual Reality
### A Comparative Study of Muscle Memory Acquisition
**Author:** Riley Nelson
**Course:** CS 465
**Institution:** Colorado State University
**Semester:** Spring 2026

---

## Project Overview
This project compares fine motor muscle memory acquisition in a Virtual Reality setting versus equivalent real-world practice. Participants learn a pattern-drawing task inspired by smartphone pattern-lock interaction in either a VR classroom environment or a physical classroom, then are assessed on accuracy and completion time in a final real-world transfer test.

## Prototype
The VR prototype is built in Unity and deployed on a Meta Quest 3. Key features include:

- 3x3 dot grid on a virtual whiteboard
- Raycast-based dot selection using right controller trigger
- Hold-and-drag pattern drawing (phone lock screen style)
- Pattern validation with green/red feedback
- 3-2-1 countdown timer triggered by a begin button
- Live stats display (correct/incorrect count)
- Timed experiment — records time to complete 3 correct attempts
- CSV data export to local file

## How to Run
1. Open project in Unity 6
2. Open `SampleScene`
3. Build and deploy to Meta Quest 3 via Android Build Settings
4. Or run in editor with Quest connected via Meta Link
